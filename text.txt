Common Emojis:
✨ feat: A new feature.
🐛 fix: A bug fix.
📝 docs: Documentation changes.
🛠 refactor: Code refactoring.
⭐ Performance improvement.
🔥 Removing code or files.
✅ test: Adding or updating tests.
🔧 build: Build system or external dependencies.
